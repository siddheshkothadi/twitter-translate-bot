===========================================
Use this template to create your .env file
===========================================

TWITTER_USERNAME= Your or bot's username with '@' symbol as a prefix
ID= Your twitter id which uniquely identifies your account
BEARER_TOKEN= Bearer Token generated for Twitter API v2 essential access
ATLAS_URI= Mongo DB atlas URI
RAPID_API_HOST= Rapid API host (that you want to use for translation)
RAPID_API_KEY= Rapid API key (generated after subscribing to the translation api)
API_KEY= API key or the consumer key generated by twitter
API_KEY_SECRET= API key secret or consumer key secret generated by twitter
ACCESS_TOKEN= Access token generated by twitter
ACCESS_TOKEN_SECRET= Access token secret generated by twitter
